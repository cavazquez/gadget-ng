# Phase 157 — Materia oscura auto-interactuante (SIDM)

**Fecha:** 2026-04-23  
**Crate afectado:** `gadget-ng-tree`  
**Archivos nuevos/modificados:** `sidm.rs`, `lib.rs`  
**Engine:** hook `maybe_sidm!` en `engine.rs`

## Resumen

Implementación del scattering elástico isótropo de SIDM. Para cada par de partículas de DM dentro del radio `2h`, se sortea un scattering con probabilidad `P = (σ/m) × ρ_local × v_rel × dt`. El scattering conserva exactamente momento y energía cinética.

## Física implementada

### Probabilidad de scattering

```text
P = (σ/m) × ρ_local × v_rel × dt
```

### Scattering elástico en sistema CM

1. Calcular velocidades del CM
2. Generar dirección aleatoria isótropa
3. Calcular nuevas velocidades conservando v_rel
4. Aplicar cambios acumulados (no durante el loop)

## API pública

| Función/Struct | Descripción |
|----------------|-------------|
| `SidmParams { sigma_m, v_max }` | Parámetros del modelo |
| `scatter_probability(v_rel, rho, sigma_m, dt)` | Probabilidad de scattering |
| `apply_sidm_scattering(particles, params, dt, seed)` | Scattering de todos los pares |

## Configuración TOML

```toml
[sidm]
enabled  = true
sigma_m  = 1.0e-5   # cm²/g en unidades internas
v_max    = 1.0e6    # corte de velocidad
```
