# Phase 130 — Polvo Intersticial Básico

**Fecha:** 2026-04-23  
**Estado:** ✅ COMPLETADA  
**Tests:** 6/6 pasados

## Objetivo

Implementar un módulo básico de polvo intersticial con dos procesos físicos: acreción de metales del gas (D/G crece) y destrucción por sputtering térmico (D/G decrece a T > T_destroy).

## Modelo físico

### Acreción (T < T_destroy)

```
dD/G/dt = Z × (d_to_g_max × Z - D/G) / τ_grow
```

La relación D/G crece hacia el máximo solar `d_to_g_max × Z` con tiempo característico `τ_grow`. Es proporcional a Z: gas sin metales no puede formar polvo.

### Sputtering térmico (T ≥ T_destroy)

```
D/G → D/G × exp(−dt / τ_sputter)
```

donde `τ_sputter = τ_grow × (T_destroy/T)²`. El sputtering es más rápido a temperaturas más altas (tasa ∝ T²).

## Implementación

### Campo en `Particle`

```rust
pub dust_to_gas: f64,  // D/G ∈ [0, d_to_g_max], default 0.0
```

### `DustSection` (config.rs)

```toml
[sph.dust]
enabled     = true
d_to_g_max  = 0.01   # solar D/G máximo
t_destroy_k = 1e6    # temperatura de sputtering en K
tau_grow    = 1.0    # tiempo de crecimiento en unidades internas
```

### `update_dust` (dust.rs)

```rust
pub fn update_dust(particles: &mut [Particle], cfg: &DustSection, gamma: f64, dt: f64)
```

1. Convierte `u → T` usando `u_to_temperature(u, gamma)`.
2. Si `T < t_destroy_k`: acreción.
3. Si `T ≥ t_destroy_k`: sputtering.
4. Clampea `dust_to_gas ∈ [0, d_to_g_max]`.

### Hook en engine.rs

```rust
// Phase 130: antes de apply_cooling
if cfg.sph.dust.enabled {
    gadget_ng_sph::update_dust(&mut local, &cfg.sph.dust, cfg.sph.gamma, cfg.simulation.dt);
}
```
