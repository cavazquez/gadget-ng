# Phase 94 — Estadísticas de la línea de 21cm

**Fecha:** 2026-04-23

## Objetivo

Implementar estadísticas de la señal de 21cm del hidrógeno neutro para el análisis in-situ:
- Temperatura de brillo δT_b por partícula
- Power spectrum P(k)₂₁cm (mK²)
- Integración en `InsituResult` y config TOML

## Nuevos archivos

### `crates/gadget-ng-rt/src/cm21.rs`

Módulo con la física completa de la señal 21cm:

**Structs**:
- `Cm21Params { t_s_kelvin, nu_21cm_mhz }` — parámetros de la línea
- `Cm21PkBin { k, delta_sq }` — bin del power spectrum Δ²₂₁(k) [mK²]
- `Cm21Output { z, delta_tb_mean, delta_tb_sigma, pk_21cm }` — salida completa

**Funciones**:
- `brightness_temperature(x_hii, overdensity, z, params)` — δT_b [mK] por partícula
  usando δT_b ≈ 27 x_HI (1+δ) √((1+z)/10) mK
- `compute_delta_tb_field(particles, chem_states, z, params)` — campo δT_b completo
- `compute_cm21_output(...)` — media, σ, y P(k)₂₁cm por CIC + PS esférico

## Archivos modificados

### `crates/gadget-ng-rt/src/lib.rs`
- `pub mod cm21;`

### `crates/gadget-ng-cli/src/insitu.rs`
- `InsituResult`: nuevo campo `cm21: Option<gadget_ng_rt::Cm21Output>`
- Cálculo de `cm21_out` cuando `cfg.cm21_enabled == true`

## Tests

5 tests unitarios en `cm21.rs`:
- `delta_tb_zero_at_full_ionization`: x_HII=1 → δT_b=0
- `delta_tb_positive_before_reionization`: x_HII=0, z=9 → δT_b≈27 mK
- `delta_tb_scales_with_overdensity`: δT_b lineal en overdensity
- `compute_cm21_output_basic`: corrida con 8 partículas mixtas
- `compute_cm21_output_empty`: lista vacía → salida vacía

Todos pasan: `cargo test -p gadget-ng-rt --lib cm21`

## Configuración TOML
