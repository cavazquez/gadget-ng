## Objetivo

Implementar la extensión especial-relativista de la MHD (SRMHD), aplicable a jets AGN
(v ~ 0.3–0.9c) y plasma ultra-caliente en halos masivos de ICM.

## Física

### Variables Conservadas (SRMHD)

En el marco del laboratorio:

```
D   = γ ρ                                    (densidad de bariones)
Sⁱ  = γ²(ρh + B²)vⁱ − γ(v·B)Bⁱ           (momento)
τ   = γ²(ρh + B²) − P − B²/2 − D           (energía)
Bⁱ  = campo magnético (marco lab)
```

donde `γ = 1/√(1−v²/c²)` es el factor de Lorentz y `h = 1 + ε + P/ρ` la entalpía relativista.

### Primitivización Newton-Raphson

El paso de conservadas → primitivas `(ρ, v, P, B)` requiere resolver:

```
f(ξ) = ξ − D h γ − B²(1 + v²/c²)/2 = 0
```

en la variable `ξ = Dhγ`, iterando hasta convergencia con tolerancia `|Δξ/ξ| < 10⁻¹⁰`.

## Implementación

### Archivo nuevo: `crates/gadget-ng-mhd/src/relativistic.rs`

- **`lorentz_factor(vel, c) → f64`**: `γ = 1/√(1−v²/c²)`; retorna ∞ si v ≥ c
- **`srmhd_conserved_to_primitive(d, s, tau, b, gamma_ad, c) → Option<(ρ, v, P)>`**:
  Primitivización Newton-Raphson (máximo 50 iteraciones)
- **`advance_srmhd(particles, dt, c, v_threshold)`**: Correcciones relativistas solo
  para partículas con `|v|/c > v_threshold`
- **`em_energy_density(b) → f64`**: Densidad de energía EM `= B²/2`

### Modificaciones de configuración

- `MhdSection.relativistic_mhd: bool` (default: `false`)
- `MhdSection.v_rel_threshold: f64` (default: `0.1`)
